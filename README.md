## Instalation

Using `Lazy.nvim`:

```lua
{'myagko/nymph.nvim'}
```

## Usage

### Lua

Add this line to your `init.lua` file:<br>
```lua
vim.cmd('colorscheme nymph')
```

### Vimscript

Add this line to your `init.vim` file (for Neovim) or `.vimrc` (for Vim):<br>
```vim
colorscheme nymph
```
